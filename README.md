#### 一些处理
+ 键入选择时大小写不受限制

+ 采取clock的继承，考虑更多属性的可能，如shape

+ 时间上的选择：
    1. hotel初始化未设置时间
    
    2. 当选择查看所有时钟时间时，clocks自动初始化并显示当前时间情况
    
    3. 未设置手机时间是查看告知用户异常
    
    4. 设置手机时间方式两种：键入`a`则采取系统获取的当前时间；键入格式为
    `yyyy mm dd HH mm ss`的时间则设置为键入时间
    
    5. 设置手机时间后clocks时间自动被设置
